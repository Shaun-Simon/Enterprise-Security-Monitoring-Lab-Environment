# Endpoint Monitoring

This section documents the deployment of endpoint visibility tooling across the enterprise security monitoring lab.

The objective of this phase was to introduce endpoint telemetry collection across Windows systems to improve visibility into security-relevant activity and prepare the environment for ingestion and centralised monitoring through a SIEM platform.

## Architecture

Sysmon provides detailed endpoint activity logging, allowing security analysts to investigate process execution, network activity, and file changes.
